# Copywriting

## Objetivo
Técnicas de copywriting e escrita persuasiva - headlines, CTAs, estruturas de copy, gatilhos mentais, landing pages, emails de venda.

## Quando usar
- Ao criar landing pages, emails e CTAs.
- Ao melhorar conversão com estrutura de copy.
- Ao testar headlines e argumentos.

Técnicas de escrita persuasiva que converte leitores em clientes.

## Princípio Central

**Ninguém liga para seu produto. Ligam para como ele melhora a vida deles.**

---

## Estruturas de Copy

### AIDA

```
ATTENTION → INTEREST → DESIRE → ACTION

1. Atenção: Headline que para o scroll
2. Interesse: "Isso é pra mim?"
3. Desejo: "Eu quero isso"
4. Ação: "O que faço agora?"
```

### PAS

```
PROBLEM → AGITATE → SOLUTION

1. Problema: Nomear a dor
2. Agitação: Intensificar a dor
3. Solução: Apresentar a saída
```

**Exemplo:**
```markdown
Você perde horas toda semana em tarefas manuais.

Enquanto isso, seus concorrentes automatizaram tudo.
Cada hora perdida é cliente perdido, dinheiro na mesa.

[Produto] automatiza em 5 minutos o que leva 5 horas.
```

### BAB

```
BEFORE → AFTER → BRIDGE

1. Before: Situação atual (dor)
2. After: Situação ideal (prazer)
3. Bridge: Como chegar lá (produto)
```

### PASTOR

```
P - Problem (problema)
A - Amplify (amplificar consequências)
S - Story (história relacionável)
T - Transformation (transformação)
O - Offer (oferta)
R - Response (chamada para ação)
```

## Headlines

### Fórmulas

| Tipo | Fórmula | Exemplo |
|------|---------|---------|
| How-to | Como [resultado] sem [objeção] | Como dobrar vendas sem contratar |
| Lista | [N] formas de [benefício] | 7 formas de economizar tempo |
| Pergunta | Você [problema/desejo]? | Você ainda perde tempo com planilhas? |
| Comando | Pare de [dor] | Pare de perder clientes |
| Segredo | O que [experts] não contam sobre [tema] | O que gurus não contam sobre tráfego pago |
| Urgência | [Resultado] em [tempo] | Inglês fluente em 90 dias |

### Palavras de Poder

**Urgência**: agora, hoje, imediato, última chance, antes que
**Exclusividade**: secreto, revelado, insider, poucos sabem
**Facilidade**: simples, fácil, rápido, sem esforço, automático
**Novidade**: novo, finalmente, descoberta, revolucionário
**Segurança**: garantido, comprovado, testado, sem risco
**Economia**: grátis, economize, lucre, ganhe, desconto

## CTAs

### Fórmulas de CTA

```
[Ação] + [Benefício]

✅ "Começar a economizar tempo"
✅ "Garantir minha vaga"
✅ "Quero dobrar minhas vendas"
✅ "Sim, quero [benefício]"
```

### Redutores de Ansiedade

Coloque abaixo do CTA:

```
✓ Sem cartão de crédito
✓ Cancele quando quiser
✓ Garantia de 30 dias
✓ Leva menos de 2 minutos
✓ Seus dados estão seguros
```

### Hierarquia de CTAs

```
[CTA Primário] - Ação principal, destaque visual
[CTA Secundário] - "Ver demo", "Saiba mais"
[CTA Terciário] - Links de texto
```

## Gatilhos Mentais

### Escassez

```markdown
❌ "Compre agora"
✅ "Apenas 7 vagas restantes"
✅ "Oferta válida até sexta-feira"
✅ "Últimas unidades em estoque"
```

### Prova Social

```markdown
❌ "Muitos clientes satisfeitos"
✅ "12.847 empresas já usam"
✅ "Avaliação 4.9/5 com 2.300 reviews"
✅ "Usado por Google, Spotify e Netflix"
```

### Autoridade

```markdown
✅ "Recomendado por [Expert]"
✅ "Certificado por [Instituição]"
✅ "Publicado em Forbes, TechCrunch"
✅ "10 anos de experiência em [área]"
```

### Reciprocidade

```markdown
Dê valor ANTES de pedir:
- Conteúdo gratuito de qualidade
- Ferramenta grátis útil
- Diagnóstico sem compromisso
- Amostra ou trial
```

### Ancoragem

```markdown
De R$997 por apenas R$297
(Você economiza R$700)

Plano anual = 12 meses pelo preço de 8
```

## Benefícios vs Features

### Transformação

```
Feature → Benefício → Benefício do Benefício

"256-bit encryption"
→ "Seus dados protegidos"
→ "Durma tranquilo sabendo que está seguro"
```

### Framework "Para que você possa..."

```
[Feature] para que você possa [benefício]

"Dashboard em tempo real para que você possa
tomar decisões sem esperar relatórios"
```

### Lista de Benefícios

```markdown
✅ [Benefício tangível]
✅ [Benefício emocional]
✅ [Benefício de status]

Exemplo:
✅ Economize 10 horas por semana (tangível)
✅ Sem stress de tarefas manuais (emocional)
✅ Pareça organizado para seus clientes (status)
```

## Objeções Comuns

| Objeção | Como quebrar |
|---------|--------------|
| "É caro" | Mostrar ROI, comparar com alternativas, parcelar |
| "Não tenho tempo" | Mostrar que economiza tempo, setup rápido |
| "Não sei se funciona pra mim" | Garantia, cases similares, trial grátis |
| "Preciso pensar" | Urgência legítima, bônus por decisão rápida |
| "Não confio" | Prova social, garantia, autoridade |

## Voz e Tom

### Conversacional

```
❌ "A plataforma disponibiliza funcionalidades avançadas"
✅ "Você ganha ferramentas poderosas sem complicação"

❌ "Nossos clientes reportam aumento de produtividade"
✅ "Clientes economizam 4 horas por semana"
```

### Segunda Pessoa

```
❌ "Os usuários podem criar..."
✅ "Você cria..."

❌ "As empresas que usam..."
✅ "Quando você usa..."
```

### Ativo > Passivo

```
❌ "Resultados serão alcançados"
✅ "Você alcança resultados"

❌ "Seu tempo será economizado"
✅ "Economize seu tempo"
```

## Formatação para Scan

### Hierarquia Visual

```markdown
# Headline (promessa principal)

Subheadline curta expandindo.

**Ponto importante em negrito.**

- Benefício 1
- Benefício 2
- Benefício 3

> Depoimento ou citação destacada

[CTA com ação clara]
```

### Regras de Legibilidade

- Parágrafos de 2-4 linhas
- Uma ideia por parágrafo
- Bullets para listas de 3+
- Negrito para pontos-chave
- Espaço em branco generoso

---

## Teste do "E Daí?"

Para cada afirmação, pergunte "E daí?" até chegar no benefício real.

```
"Nossa plataforma é rápida"
→ E daí?
"Você economiza tempo"
→ E daí?
"Faz mais em menos tempo"
→ E daí?
"Sai do trabalho mais cedo, fica com a família"
→ Benefício emocional real!
```

## Checklist de Copy

### Antes de Publicar

- [ ] Headline prende em 3 segundos?
- [ ] Benefício claro no início?
- [ ] Linguagem do cliente (não jargão)?
- [ ] Números específicos?
- [ ] Objeções quebradas?
- [ ] Prova social presente?
- [ ] CTA claro com benefício?
- [ ] Escaneável (bullets, negrito)?
- [ ] Urgência ou escassez?
- [ ] Passou no teste "E daí?"
