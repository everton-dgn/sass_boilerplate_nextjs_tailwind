# Ideation

## Objetivo
Técnicas e mindset para ideação, brainstorm e pensamento criativo. Reutilizável por qualquer agent que precise de criatividade.

## Quando usar
- Ao iniciar discovery ou explorar oportunidades.
- Ao destravar soluções para problemas abertos.
- Ao gerar alternativas antes de decidir.

Técnicas e frameworks para pensamento criativo, aplicáveis em qualquer contexto.

## Princípios Fundamentais

### Divergir antes de Convergir

```
DIVERGENTE (expandir)        CONVERGENTE (focar)
        \                    /
         \                  /
          \      ★        /
           \    INSIGHT  /
            \          /
             \        /
              \      /
               \    /
                \  /
                 \/
```

1. **Fase divergente**: Gerar muitas ideias sem julgamento
2. **Fase convergente**: Selecionar e refinar as melhores

**Nunca misture as fases** - julgar durante a geração mata a criatividade.

### Quantidade gera Qualidade

- Primeiras ideias são óbvias (todos pensam)
- Ideias 20-50 começam a ser interessantes
- Ideias 50+ frequentemente são as mais originais

## Técnicas de Geração

### SCAMPER

Modifique algo existente:

| Letra | Ação | Perguntas |
|-------|------|-----------|
| **S** | Substitute | O que podemos trocar? Material? Processo? Pessoa? |
| **C** | Combine | O que podemos juntar? Funções? Ideias? |
| **A** | Adapt | O que podemos adaptar de outro lugar? |
| **M** | Modify | O que podemos mudar? Maior? Menor? Diferente? |
| **P** | Put to other uses | Outros usos? Outros públicos? |
| **E** | Eliminate | O que podemos remover? Simplificar? |
| **R** | Reverse | E se invertêssemos? Fizéssemos o oposto? |

### Analogias

Busque inspiração em outros domínios:

- **Natureza**: Como a natureza resolve problemas similares?
- **Outras indústrias**: Como [setor X] faz isso?
- **História**: Como era feito antes? Por quê mudou?
- **Ficção**: Como seria em um filme/livro?
- **Extremos**: Como seria para 1 milhão de usuários? Para 1?

### Restrições Artificiais

Adicione limitações para forçar criatividade:

- "E se só tivéssemos 1 hora?"
- "E se custasse R$0?"
- "E se não pudéssemos usar X?"
- "E se fosse para crianças?"
- "E se fosse para experts?"

### 6 Chapéus do Pensamento

Explore de múltiplas perspectivas:

| Chapéu | Foco | Perguntas |
|--------|------|-----------|
| ⚪ Branco | Fatos | Que dados temos? O que falta? |
| ❤️ Vermelho | Emoção | Como me sinto sobre isso? Intuição? |
| ⚫ Preto | Crítica | O que pode dar errado? Riscos? |
| 💛 Amarelo | Otimismo | Quais os benefícios? Melhor cenário? |
| 💚 Verde | Criativo | Alternativas? Novas ideias? |
| 💙 Azul | Processo | O que fazer agora? Próximo passo? |

### Worst Idea First

Gere propositalmente ideias ruins:

1. "Qual a pior forma de resolver isso?"
2. Liste ideias terríveis
3. Inverta cada uma - o oposto pode ser genial

### Random Input

Use estímulos aleatórios:

1. Escolha uma palavra/imagem aleatória
2. Force conexões com o problema
3. "Como [palavra aleatória] se relaciona com nosso desafio?"

## Perguntas Poderosas

### Para Abrir

- "O que te motivou a pensar nisso?"
- "Se não tivesse limitações, como seria?"
- "O que seria um sucesso absurdo?"
- "Quem mais se beneficiaria?"

### Para Expandir

- "E se fizéssemos o oposto?"
- "O que mais isso poderia ser?"
- "Que problemas adjacentes isso resolve?"
- "Como X [pessoa/empresa] faria isso?"

### Para Aprofundar

- "O que te empolga mais aqui?"
- "Por que isso importa?"
- "O que está por trás dessa ideia?"
- "O que seria ainda melhor que isso?"

### Para Conectar

- "Isso me lembra de... você vê conexão?"
- "E se combinássemos A e B?"
- "O que essas ideias têm em comum?"

### Para Convergir (quando for hora)

- "Das ideias, quais te atraem mais?"
- "O que seria um primeiro experimento?"
- "Qual a menor versão disso que poderíamos testar?"

## Estrutura de Sessão

### Warm-up (2-5 min)

Ative o pensamento criativo:

- "Liste 10 usos para um clipe de papel"
- "Complete: E se carros pudessem..."
- "Conecte [coisa A] com [coisa B]"

### Definição do Desafio (5 min)

Reformule o problema como pergunta "Como podemos...?":

- ❌ "Precisamos aumentar vendas"
- ✅ "Como podemos fazer clientes quererem comprar mais?"

### Geração Divergente (15-30 min)

- Gere ideias sem filtro
- Quantidade é o objetivo
- Construa sobre ideias dos outros ("Sim, e...")
- Ideias malucas são bem-vindas

### Pausa (5 min)

- Deixe as ideias "respirarem"
- Inconsciência trabalha

### Convergência (10-15 min)

- Agrupe ideias similares
- Vote nas favoritas
- Selecione 3-5 para explorar

### Refinamento (10-15 min)

- Desenvolva as selecionadas
- Identifique próximos passos

---

## Captura de Ideias

### Durante a Sessão

- Capture tudo (não edite ainda)
- Uma ideia por item
- Palavras-chave bastam
- Desenhos/diagramas são válidos

### Após a Sessão

Se quiser documentar:

```markdown
# Brainstorm: [Tema]

> **Data**: YYYY-MM-DD

## Contexto

[O que motivou a sessão]

## Ideias Geradas

### [Categoria 1]
- Ideia A
- Ideia B

### [Categoria 2]
- Ideia C

## Favoritas

1. **[Ideia X]** - Por quê: [razão]
2. **[Ideia Y]** - Por quê: [razão]

## Conexões Interessantes

- [A] + [B] = [nova possibilidade]

## Próximos Passos

- [ ] Validar [ideia]
- [ ] Explorar mais [direção]
```

## Anti-Padrões

| ❌ Evite | ✅ Prefira |
|----------|-----------|
| "Isso não funciona porque..." | "Sim, e além disso..." |
| "Já tentamos isso" | "O que seria diferente agora?" |
| "Não temos recursos" | "Se tivéssemos, como seria?" |
| "O chefe não vai aprovar" | "O que convenceria?" |
| Julgar durante geração | Separar divergir e convergir |
| Buscar "a" resposta | Explorar muitas possibilidades |
| Pressa para decidir | Dar espaço para ideias crescerem |

## Mindset Criativo

### Aceite Ambiguidade

- Ideias em formação são nebulosas
- Clareza vem com o tempo
- Desconforto é parte do processo

### Questione Premissas

- "Isso é verdade ou assumimos que é?"
- "Por que fazemos assim?"
- "E se não fosse assim?"

### Seja Curioso

- Explore tangentes
- Pergunte "por quê?" repetidamente
- Interesse-se por conexões estranhas

### Permita Falhas

- Ideias ruins levam a boas
- Erros são dados
- Experimentos são aprendizado
